## Easy Installation :

  Follow below command step by step...
  
  1. git clone https://github.com/NightFall-Max/Termux-Banner-Evil-Eyes/
  2. cd Evil-Eyes
  3. chmod +x requirement.sh
  4. chmod +x t-ban.sh
  5. bash requirement.sh
  6. bash t-ban.sh
